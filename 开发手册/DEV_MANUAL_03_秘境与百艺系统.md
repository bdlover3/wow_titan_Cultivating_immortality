### 3.5 技术实现

```lua
XiuxianBattleground = {
    ["战歌峡谷"] = { name = "灵脉峡谷", desc = "两宗争夺灵脉之地的峡谷" },
    ["阿拉希盆地"] = { name = "灵石盆地", desc = "四宗争夺灵石矿脉的盆地" },
    ["奥特兰克山谷"] = { name = "万妖山谷", desc = "两宗在妖兽横行的山谷中决战" },
    ["风暴之眼"] = { name = "风暴秘境", desc = "浮空秘境中的灵脉争夺" },
    ["远征海滩"] = { name = "远征滩头", desc = "两宗抢滩登陆的战场" },
    ["古代海滩"] = { name = "远古海滩", desc = "远古遗迹中的灵宝争夺" },
}

EventManager:Register("PLAYER_ENTERING_BATTLEGROUND", function()
    local bgName = GetRealZoneText()
    local xiuxianBg = XiuxianBattleground[bgName]
    if xiuxianBg then
        Xiuxian_SisterSay("师兄，欢迎来到" .. xiuxianBg.name .. "！" .. xiuxianBg.desc)
    end
end)

EventManager:Register("PLAYER_PVP_KILLS_CHANGED", function()
    Xiuxian_ShowToast("斗法胜绩+1，道友威武！")
end)
```

**API调用**:
- `GetPVPRankInfo(rank)` — PvP等级信息
- `UnitPVPRank("player")` — PvP等级ID
- `GetHonorCurrency()` — 荣誉点数
- `GetRealZoneText()` — 当前区域(判断战场)
- `IsInInstance()` — 是否在副本/战场中
- `GetZonePVPInfo()` — 区域PvP信息

**相关事件**:
- `PLAYER_ENTERING_BATTLEGROUND` — 进入战场
- `PLAYER_PVP_KILLS_CHANGED` — PvP击杀变化
- `PLAYER_PVP_RANK_CHANGED` — PvP等级变化
- `HONOR_CURRENCY_UPDATE` — 荣誉点数更新
- `ZONE_CHANGED_NEW_AREA` — 进入新区域(检测战场)

## 四、秘境与战场API汇总

将第2.8节(秘境系统)、第2.9节(百艺系统)、第三章(战场系统)涉及的所有API集中汇总，便于开发时快速查阅。

### 4.1 秘境系统 API

| API函数 | 用途 | 修仙映射 |
|--------|------|---------|
| `GetInstanceInfo()` | 获取当前副本信息 | → 秘境信息(名称、类型、难度) |
| `GetRealZoneText()` | 获取当前区域名称 | → 秘境/灵脉名称 |
| `GetNumSavedInstances()` | 获取副本锁定数量 | → 秘境冷却数量 |
| `GetSavedInstanceInfo(index)` | 获取副本锁定详情 | → 秘境冷却详情(剩余时间、秘境名) |

### 4.2 百艺系统 API

| API函数 | 用途 | 修仙映射 |
|--------|------|---------|
| `GetProfessions()` | 获取已学习的专业技能索引 | → 百艺列表 |
| `GetProfessionInfo(index)` | 获取专业详情(名称、等级、图标) | → 百艺详情(艺名、品阶、图标) |
| `GetTradeSkillLine()` | 获取当前打开的专业技能线 | → 当前修习的百艺 |

### 4.3 战场系统 API

| API函数 | 用途 | 修仙映射 |
|--------|------|---------|
| `GetPVPRankInfo(rank)` | 获取PvP等级信息(名称、图标) | → 斗法境界信息 |
| `UnitPVPRank("player")` | 获取玩家PvP等级ID | → 斗法境界ID |
| `GetHonorCurrency()` | 获取当前荣誉点数 | → 斗法积分 |
| `GetRealZoneText()` | 获取当前区域名称 | → 判断所在战场/秘境 |
| `IsInInstance()` | 是否在副本或战场中 | → 是否在秘境/争锋中 |
| `GetZonePVPInfo()` | 获取区域PvP状态 | → 灵脉争端状态 |

### 4.4 战场系统事件

| 事件名 | 触发时机 | 修仙用途 |
|-------|---------|---------|
| `PLAYER_ENTERING_BATTLEGROUND` | 进入战场 | 宗门争锋开启提示 |
| `PLAYER_PVP_KILLS_CHANGED` | PvP击杀数变化 | 斗法胜绩更新 |
| `PLAYER_PVP_RANK_CHANGED` | PvP等级变化 | 斗法境界突破 |
| `HONOR_CURRENCY_UPDATE` | 荣誉点数更新 | 斗法积分更新 |
| `ZONE_CHANGED_NEW_AREA` | 进入新区域 | 检测战场/秘境区域 |
