# 游戏暴露接口分析报告

## 概述

本报告基于对WoW插件代码的深入分析，总结了魔兽世界游戏暴露给插件使用的API接口。这些接口主要分为以下几个类别：

## 1. 地图相关API (C_Map)

### C_Map.GetBestMapForUnit
- **功能**：获取指定单位当前所在的最佳地图ID
- **入参**：`unit` (string) - 单位标识符，如 "player"
- **出参**：地图ID (number)
- **使用示例**：
```lua
local mapID = C_Map.GetBestMapForUnit('player')
```

### C_Map.GetPlayerMapPosition
- **功能**：获取玩家在指定地图上的位置
- **入参**：`mapID` (number) - 地图ID, `unit` (string) - 单位标识符
- **出参**：Vector2D对象，包含x和y坐标
- **使用示例**：
```lua
local mappos = C_Map.GetPlayerMapPosition(mapID, 'player')
local x, y = mappos:GetXY()
```

### C_Map.GetWorldPosFromMapPos
- **功能**：将地图坐标转换为世界坐标
- **入参**：`mapID` (number) - 地图ID, `vector` (Vector2D) - 地图坐标
- **出参**：worldPos (Vector3D) - 世界坐标位置
- **使用示例**：
```lua
local vector00 = CreateVector2D(0, 0)
local instance, worldPos = C_Map.GetWorldPosFromMapPos(mapID, vector00)
```

## 2. 聊天相关API (C_ChatInfo)

### C_ChatInfo.SendAddonMessage
- **功能**：发送插件间通信消息
- **入参**：
  - `prefix` (string) - 消息前缀/频道名
  - `message` (string) - 消息内容
  - `chatType` (string) - 聊天类型 ("PARTY", "RAID", "GUILD", etc.)
  - `target` (string, optional) - 目标玩家名
- **出参**：无
- **使用示例**：
```lua
C_ChatInfo.SendAddonMessage("PIG", "Hello World", "PARTY")
```

### C_ChatInfo.RegisterAddonMessagePrefix
- **功能**：注册插件消息前缀
- **入参**：`prefix` (string) - 消息前缀
- **出参**：无
- **使用示例**：
```lua
C_ChatInfo.RegisterAddonMessagePrefix("PIG")
```

## 3. 插件管理API (C_AddOns)

### C_AddOns.IsAddOnLoaded
- **功能**：检查指定插件是否已加载
- **入参**：`name` (string) - 插件名称
- **出参**：boolean - 是否已加载
- **使用示例**：
```lua
if C_AddOns.IsAddOnLoaded("DBM-Core") then
    -- DBM已加载
end
```

### C_AddOns.GetAddOnInfo
- **功能**：获取插件信息
- **入参**：`index` (number) - 插件索引或名称
- **出参**：name, title, notes, loadable, reason, security, newVersion
- **使用示例**：
```lua
local name, title, notes, loadable, reason = C_AddOns.GetAddOnInfo(1)
```

### C_AddOns.GetAddOnEnableState
- **功能**：获取插件启用状态
- **入参**：`account` - 账户信息, `name` (string) - 插件名称
- **出参**：状态值 (0=禁用, 1=启用, 2=加载需求)
- **使用示例**：
```lua
local state = C_AddOns.GetAddOnEnableState(account, "PIG")
```

### C_AddOns.LoadAddOn
- **功能**：动态加载插件
- **入参**：`name` (string) - 插件名称
- **出参**：loaded (boolean), reason (number) - 加载结果原因
- **使用示例**：
```lua
local loaded, reason = C_AddOns.LoadAddOn("Blizzard_ArchaeologyUI")
if not loaded then return end
```

### C_AddOns.GetAddOnMetadata
- **功能**：获取插件元数据
- **入参**：`addonName` (string) - 插件名, `variable` (string) - 变量名 (如 "Version")
- **出参**：string - 变量值
- **使用示例**：
```lua
local version = C_AddOns.GetAddOnMetadata("PIG", "Version")
```

## 4. 小地图相关API (C_Minimap)

### C_Minimap.GetNumTrackingTypes
- **功能**：获取跟踪类型数量
- **入参**：无
- **出参**：number - 跟踪类型数量
- **使用示例**：
```lua
local count = C_Minimap.GetNumTrackingTypes()
```

### C_Minimap.GetTrackingInfo
- **功能**：获取指定跟踪类型的信息
- **入参**：`index` (number) - 跟踪类型索引
- **出参**：trackingInfo table - 包含name, icon, active等字段
- **使用示例**：
```lua
for i=1, C_Minimap.GetNumTrackingTypes() do
    local trackingInfo = C_Minimap.GetTrackingInfo(i)
    if trackingInfo["name"]:find("Track Digsites") then
        -- 找到考古挖掘点跟踪
    end
end
```

### C_Minimap.SetTracking
- **功能**：设置小地图跟踪状态
- **入参**：`id` (number) - 跟踪类型ID, `on` (boolean) - 开启/关闭
- **出参**：无
- **使用示例**：
```lua
C_Minimap.SetTracking(id, true)
```

## 6. 技能相关API (C_Spell)

### C_Spell.GetSpellInfo
- **功能**：获取技能信息
- **入参**：`spellID` (number) - 技能ID
- **出参**：spellInfo table - 包含name, iconFileID, castTime, minRange, maxRange, spellID等
- **使用示例**：
```lua
local spellInfo = C_Spell.GetSpellInfo(73979)
if spellInfo then
    local name = spellInfo.name
    local iconFileID = spellInfo.iconFileID
end
```

### C_Spell.GetSpellCooldown
- **功能**：获取技能冷却信息
- **入参**：`spellID` (number) - 技能ID
- **出参**：cooldownInfo table - 包含startTime, duration, isEnabled, modRate
- **使用示例**：
```lua
local cooldownInfo = C_Spell.GetSpellCooldown(spellID)
if cooldownInfo then
    local startTime = cooldownInfo.startTime
    local duration = cooldownInfo.duration
    local isEnabled = cooldownInfo.isEnabled
end
```

### C_SpellBook.GetSpellBookSkillLineInfo
- **功能**：获取技能线信息

```lua
local skillInfo = C_SpellBook.GetSpellBookSkillLineInfo(Enum.SpellBookSkillLineIndex.MainSpec)
if skillInfo then
    local name = skillInfo.name
    local iconFileID = skillInfo.iconFileID
end
```
